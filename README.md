# TeraCreators Help

A community-created help guide for TeraCreators games, built with [Docusaurus](https://docusaurus.io/).

The site is currently focused on **Havoc Hotel 3** and organizes stable community knowledge into clear sections such as:

- Overview
- Tips
- Tasks
- FAQ
- References
- Terms

## Local Development

Install dependencies:

```bash
npm install

Start the local development server:

npm run start

This starts the site locally, usually at http://localhost:3000/TeraCreators-Help/ or http://localhost:3000/, depending on your setup.

Build

Create a production build:

npm run build

This generates the static site in the build directory.

Deployment

This site is deployed with GitHub Pages using GitHub Actions.

When changes are pushed to the main branch, GitHub Actions builds and deploys the site automatically.
