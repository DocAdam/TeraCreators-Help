# Havoc Hotel 1 Overview

![Havoc Hotel 1 cover image](/img/havoc-hotel-1/hh1-cover.jpeg)

**HAVOC HOTEL 1 [ROGUELIKE]** is a Fortnite island by **teracreators**. Its island code is **3739-0247-7626**.

This guide pulls together useful community-shared help for **Havoc Hotel 1** and rewrites it into simpler reference content.

It focuses on practical progression help, recurring answers, weapons, upgrades, rescues, and quick-reference info pulled from the original community material.

## What This Guide Covers

This guide organizes community-shared HH1 help into practical sections so players can quickly find instructions, answers, and reference material.

- **Tips** explains progression strategy and repeated gameplay advice
- **Tasks** turns common guidance into clearer step-by-step instructions
- **FAQ** answers recurring player questions
- **References** collects lookup notes and supporting screenshots
- **Terms** defines commonly used game language

## About the Island

Once a luxurious hotel, Haven Hotel was seized by a dangerous group and became Havoc Hotel. The island is built around solo or co-op roguelike progression through 18 stages of guards and bosses, with weapons, upgrades, consumables, power-ups, rescues, and hidden notes tied to the hotel’s downfall.
