# Raptor Heist Overview

**Raptor Heist: A Havoc Hotel Roguelike** is a Fortnite island by **teracreators**. Its island code is **1311-3035-1480**.

This guide pulls together useful community-shared help for **Raptor Heist** and rewrites it into simpler reference content.

## About the game

Raptor Heist is part of the broader Havoc Hotel-style lineup from **teracreators / Teravision Games**.

The official game page emphasizes:

- stealing and protecting a fragile Dino Egg
- escaping with it successfully
- upgrading through Havoc Hotel-style progression
- playing with up to 5 friends
