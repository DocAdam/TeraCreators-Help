# Terms

These are common terms and repeated phrases that show up in HH2 help discussions.

## Crown

A long-term upgrade track that community replies describe as progressing through named stages such as Wood, Bronze, Silver, Gold, Diamond, and Ruby.

## Escape time

Your completion time for a run, usually used for leaderboard submissions.

## Harvesting tool

The tool players were told to use when trying to claim certain room-clear reward pickups.

## Prestige

A reset system that community members describe as mostly giving bragging-right value in HH2, with some progression being reset and prices increasing again.

## Private lobby

A fresh session created separately from the current bugged or stuck session. Community messages suggest this can fix some trigger-related issues.

## Room-clear reward

A reward that appears after successfully finishing a room.

## Swarmstrike

A reported reward tied to one of the community-shared HH2 codes.
