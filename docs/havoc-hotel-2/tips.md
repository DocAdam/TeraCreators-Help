# Havoc Hotel 2 Tips

## Understanding HH2 Codes

HH2 does appear to have secret codes, but community messages show some confusion over when they were added and whether all reported codes still work.

### Key points

- Early replies said HH2 did not have codes
- Later community posts shared specific HH2 codes and rewards
- Some players later reported that some codes stopped working
- Treat reported codes as community-sourced, not guaranteed

### Quick reference

- Community-reported HH2 codes:
  - `127946` — 300 million
  - `190398` — Swarmstrike
  - `954628` — 100 million

### Notes

- Some messages suggest there may be additional hidden codes near the start of the map or after the Rocket Launcher boss, but no confirmed value was captured in the shared resource
- Some players later said some codes had stopped working
- Swarmstrike appears to be treated like a limited-use reward item in community discussion

## Understanding Rest Area and Session Issues

Some recurring HH2 issues appear tied to a specific game session rather than your account.

### Key points

- If the rest area or a trigger stops working, the problem may remain for that current session
- Leaving and returning to the same session may not fix it
- Starting a fresh private lobby was reported as a practical workaround
- A working session may show the opening cutscene again

### Quick reference

- Try a fresh private lobby when progress triggers stop working
- Use this as a workaround, not as a permanent fix

### Notes

- This is a recurring workaround, but still community-reported

## Understanding HH2 Versus HH3 Systems

A community reply said HH2 and HH1 are broadly similar in structure, but HH2 does not use Soap collection and investment the way HH3 does, and it was described as much easier overall.

### Important

- Do not assume HH3 systems apply directly to HH2
- Code behavior, prestige value, and progression details may differ

### Notes

- This is a high-level comparison, not a full system breakdown
