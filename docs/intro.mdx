## Start Here

If you are not sure where to begin:

- open **Overview** for the game you are playing
- use **Tasks** for step-by-step help
- use **Tips** for strategy and progression advice
- use **References** for quick lookups
- use **Terms** for definitions and shorthand
- use **FAQ** for repeated questions where that section exists
- use **Rescues** for hostage-related help in **Havoc Hotel 3**

## How the Content Is Organized

- **Overview** explains what the guide covers and how to use it
- **Tips** focuses on strategy and practical advice
- **Tasks** turns repeated guidance into clear steps
- **FAQ** answers common repeat questions where needed
- **References** collects lookup information such as rooms, rewards, milestones, and screenshot-based reference points
- **Terms** defines commonly used game language and shorthand
- **Rescues** collects hostage locations, duplicate rescue steps, and rescue-specific notes for **Havoc Hotel 3**

## Source Context

These guides are built from community-shared help and reorganized into cleaner docs so useful information is easier to browse and reuse.
