# TeraCreators Help

This site organizes community-shared game knowledge into clear, reusable guide pages.

It focuses on stable gameplay help, repeated player guidance, and quick-reference information that is easier to browse than scattered chat messages.

## How the Content Is Organized

- **Rescues** collects hostage locations, duplicate rescue steps, and rescue-specific notes
- **Overview** explains what the guide covers and how to use it
- **Tips** focuses on strategy and practical advice
- **Tasks** turns repeated guidance into clear steps
- **FAQ** answers common repeat questions
- **References** collects lookup information such as rooms, codes, and boss notes
- **Terms** defines commonly used game language and shorthand
