## About the game

The official game page describes Havoc Hotel 3 as a brutal roguelike built around:

- solo and co-op play
- 30 difficult rooms
- earning by defeating enemies
- unlocking weapons and permanent power-ups
- investing soap to improve weapon and booster chances
- choosing a random reward after each room
