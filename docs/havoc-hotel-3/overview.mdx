# Havoc Hotel 3 Help Guide

This guide brings together useful community knowledge and turns it into clear, practical help content that can be shared with other players.

It focuses on stable gameplay information, common questions, step-by-step instructions, and quick-reference material.

## What This Guide Covers

This guide organizes community-shared help content into practical sections so players can quickly find instructions, answers, and reference material.

### What each section is for

- **Tips** explains systems and strategies
- **Tasks** provides step-by-step instructions
- **FAQ** answers common repeat questions
- **References** collects lookup information such as codes, rooms, and bosses
- **Terms** defines commonly used words and shorthand that appear throughout the guide, such as <Term name="AFK">AFK</Term>, <Term name="Prestige">prestige</Term>, <Term name="Loot pool">loot pool</Term>, and <Term name="Rescue">rescue</Term>

### What is excluded

- Transitory issues
- Bug reports
- One-off complaints
- Unverified edge-case claims that are not repeated elsewhere

## Quick Navigation

- New player: start with [FAQ](./faq)
- Need steps: go to [Tasks](./tasks)
- Need hostage locations or duplicate rescue steps: go to [Rescues](./rescues)
- Need strategy: go to [Tips](./tips)
- Need exact values or locations: use [References](./references)
- Need definitions: check [Terms](./terms)

## Notes About the Guide

- This guide is based on community-shared information
- Some strategies are player-tested rather than official explanations
- Codes and rewards may change over time
