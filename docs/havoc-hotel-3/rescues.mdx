- Players report that duplicate rescues can push the displayed total above that, including counts like **35/20**
- <Term name="Rescue">Rescue</Term> rewards become more useful as your rescue value increases

## Duplicate Rescue Method

This is the most commonly repeated <Term name="Rescue">rescue</Term> trick.

### Steps

1. <Term name="Rescue">Rescue</Term> the hostage normally.
2. Immediately accept the room reward.
3. Return to the same hostage location.
4. <Term name="Rescue">Rescue</Term> the hostage again.

### Important

- This is community-reported and appears to work on most rescues, but not every rescue
- Players report this can raise the visible rescue count beyond the normal total
